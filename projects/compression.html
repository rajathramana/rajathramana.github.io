High-Throughput Heterogeneous Compression Engine

This project focuses on designing a real-time data compression pipeline capable of sustaining 400 Mb/s throughput on an Ultra96 SoC by leveraging heterogeneous compute resources. The goal was to minimize latency while efficiently utilizing CPU, SIMD, and FPGA fabric.

System Architecture

The pipeline partitions computation across multiple processing elements to eliminate bottlenecks:

  • ARM Cortex-A53: control logic, buffering, and data orchestration
  • NEON intrinsics: vectorized preprocessing and symbol preparation
  • FPGA fabric: hardware-accelerated LZW compression

Implementation Details

  • Designed a streaming architecture to avoid memory stalls
  • Used DMA-based transfers to overlap computation and I/O
  • Implemented hardware/software synchronization using OpenCL

Verification & Results

End-to-end verification was performed using RTL testbenches and host-device integration. The final system achieved sustained throughput without packet loss under continuous Ethernet streaming.
